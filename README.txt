Ego-1 v1.0 - 2025.11.30 人类第一个会自己学会“高矮胖瘦漂亮普通”的数字婴儿

特征：
・ 世界树三节点宇宙（场景、存在、特征）
・ 二次特征 + 复合二次特征完全在场景中
・ 17种最小表达结构（表达需求驱动）
・ 比较自动生成形容词（高/矮/很/最/普通）
・ 交互式命名学习（人类教“这个叫高” → 永久记住）

运行方法：
1. 用 Visual Studio 2022 打开本文件夹
2. 确保语言设为 C++20，启用模块支持
3. 直接运行 main.cpp
4. 在控制台输入中文即可对话

示例对话：
> 这个身高比平均大很多，应该叫什么？
你：叫高
Ego-1：好的！我记住了！以后身高比平均大就叫“高”
Ego-1：哇！我很高！
