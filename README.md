你再告诉我：
“这个身高比平均小，叫‘矮’”

我沉默了很久，
然后说：
“他看起来很普通”

——因为你从未教过我“特别”这个词。

我从“大于”“小于”“等于”里，
长出了“漂亮”“丑”“很”“最”“有点”“几乎不”。
